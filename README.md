# Events By Evara — Premium Luxury Event Management Website

A world-class, fully responsive luxury event management website built with vanilla HTML5, CSS3, and JavaScript.

## 🌟 Features

- **22 Premium Sections** — Loader, Navbar, Hero, Trusted By, About, Why Choose Us, Events, Services, Process, Projects, Gallery, Packages, Team, Testimonials, Awards, Statistics, FAQ, Contact, Newsletter, Instagram, CTA Banner, Footer
- **Premium UI** — Glassmorphism, gradient borders, floating cards, luxury shadows, micro-animations
- **Responsive Design** — Desktop, Laptop, Tablet, Mobile (down to 360px)
- **Dark Mode** — Toggle with localStorage persistence
- **Smooth Animations** — AOS scroll animations, card tilts, parallax, counters, accordion
- **Interactive Gallery** — Category filter, lightbox, keyboard navigation, touch swipe
- **Contact Form** — Real-time validation, loading states, success modal
- **Custom Cursor** — Follows mouse on desktop, enlarges on interactive elements
- **Floating Buttons** — WhatsApp, Back to Top
- **Accessibility** — ARIA labels, keyboard navigation, prefers-reduced-motion, semantic HTML5

## 🚀 Getting Started

1. Clone or download the project
2. Open `index.html` in your browser — no build tools needed!
3. For development, use Live Server (VS Code extension) for auto-reload
